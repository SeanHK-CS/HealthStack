# HealthStack

## Why I built this

I realized I'd gained weight and wasn't as fit as I used to be. I wanted to get healthier, but I kept bouncing between scattered tabs — one app for exercises, another for supplements, another for calorie math. HealthStack is the reference I wished I had: workouts, evidence-ranked supplements, and nutrition basics in one place, offline, no accounts.

## What it is

Personal fitness reference: 873 exercises, 33 evidence-ranked supplements, macro calculator + food table, and a built-in coach chat (ask "optimal upper body workout").

## Run it locally
1. Unzip.
2. Double-click `index.html`. That's it (works offline; exercise photos need internet).

## Tests (optional, needs Node)
    npm install jsdom
    node test/unit.test.js
    node test/coach.test.js
    node test/smoke.test.js

Data: exercises from free-exercise-db (MIT-style open license); supplement tiers curated from Examine.com / ISSN consensus; food macros are approximate USDA values. Not medical advice.
